# cantopop-lyrics

> **Note**
>  Only support for popular Cantopop songs.

## Installation

```shell
git clone git@github.com:ngseke/cantopop-lyrics.git
cd cantopop-lyrics
npm i
```

## Usage

```shell
npm start
```

or

```shell
npm start [keyword]
```

### e.g.

```
npm start 無盡
npm start 陳奕迅\ 浮誇
npm start 到此為止\ dear\ jane
```
